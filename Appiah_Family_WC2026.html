1
Go to emailjs.com → create a free account
2

3
Create an Email Template using the variables below → copy your Template ID
4
Go to Account → API Keys → copy your Public Key
5
Paste all three below and tap Save
Template variables to use in EmailJS:
{{to_email}} — recipient email
{{to_name}} — recipient name
{{subject}} — email subject
{{message}} — leaderboard / update body
